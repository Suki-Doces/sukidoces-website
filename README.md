# Suki Doces Website (v0.5)

A charming and modern landing page designed for **Suki Doces**, a confectionery shop specialized in sweets that create memorable moments. This project focuses on a clean user interface with soft aesthetics and smooth interactivity.

## ✨ Features

* **Interactive Carousel**: A custom-built banner slider showcasing promotions like "50% OFF" and "Free Shipping".
* **Responsive Design**: Styled with a "Inter" typography and a pink-themed color palette for a professional look.
* **Sticky Navigation**: A mobile-friendly bottom menu for quick access to Home, Search, Favorites, Cart, and Profile.
* **Product Showcase**: Sections for "Best Sellers", "Trending", and "New Arrivals".
* **Business Info**: Dedicated sections for delivery information, contacts, and store location.

## 🛠️ Technologies Used

* **HTML5**: Semantic structure.
* **CSS3**: Custom properties (variables), Flexbox, and Grid layout.
* **JavaScript (Vanilla)**: Custom logic for the peek-a-boo carousel effect and automated transitions.
